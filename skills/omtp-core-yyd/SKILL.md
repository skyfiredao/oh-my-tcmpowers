# 圆运动核心理论

本技能是圆运动 skill 体系的**核心理论**。包含升降浮沉圆运动模型、中气轴心、五脏气机偏性、"以偏纠偏"总则。不作为入口直接使用，由 omtp-agent-PengZiYi 前置加载。

来源：彭子益《圆运动的古中医学》

## 圆运动模型

### 二十四节气圆运动

大气的圆运动即自然界阳气的升降浮沉循环：

```
        夏至（浮·极）
         ↑ 火
    立夏 / \ 立秋
       /   \
春分 木      金 秋分
（升·极）    （降·极）
       \   /
    立春 \ / 立冬
         ↓ 水
        冬至（沉·极）

        中央：土（中气·轴）
```

- **升**（春）：阳气由下而上，木气主之
- **浮**（夏）：阳气外达最盛，火气主之
- **降**（秋）：阳气由上而下，金气主之
- **沉**（冬）：阳气内收最深，水气主之
- **中**（长夏）：土气居中为轴心，脾升胃降，旋转不息

### 圆运动一句话

> 降→沉→升→浮，中气为轴，轴运轮行，轮运轴灵。

降不下去则浮不起来；升不上去则沉不下去。任一环节阻滞则圆运动破坏。

### 中气为轴心

中气者，脾胃也。脾主升清，胃主降浊。脾升则木火得升，胃降则金水得降。中气旋转如轮之有轴，轴不运则轮不行。

**核心命题**：一切内伤杂病，皆中气不足、升降失常之变化。

### 中气与四维关系

| 关系 | 说明 |
|------|------|
| 中气足 → 升降正常 | 木升火浮金降水沉，各归其位 |
| 中气虚 → 升降失常 | 木不升则郁，火不降则上炎，金不降则燥，水不升则寒 |
| 治中气即治四维 | 补中气可使升降复常，不必一味追逐末端症状 |

## 五脏气机偏性

### 各脏升降属性

| 脏腑 | 气机方向 | 正常状态 | 失常状态 |
|------|----------|----------|----------|
| 肝（木） | 升 | 升而不郁，疏泄调达 | 升不及→郁；升太过→风、动 |
| 心（火） | 浮 | 浮而能敛，宣通不散 | 浮不敛→上炎；浮不及→寒凝 |
| 脾（土） | 升清 | 脾气上升，运化水谷 | 升不及→泄泻、下坠 |
| 胃（土） | 降浊 | 胃气下降，腐熟传导 | 降不及→呕逆、痞满 |
| 肺（金） | 降 | 降而收敛，清肃下行 | 降不及→咳逆、气喘 |
| 肾（水） | 沉 | 沉而封藏，温养根本 | 藏不及→遗泄、浮阳 |

### 相火与升降

胆经相火，随三焦下降而归肾。若胆经不降，相火逆升，上扰心肺、下耗肾水。

> 相火不降，百病丛生。

降相火之法：非苦寒直折，乃降胆经使相火归根。

## "以偏纠偏"总则

### 核心原理

> 人之生也，得大气五行圆运动之**全**；物之生也，得大气五行圆运动之**偏**。故物为人身之药。

- 每味药物禀五行某一偏性
- 以药物之偏性纠正人体气机之偏差
- 升药治降不及，降药治升太过，补中气治轴心不运

### 用药维度

| 维度 | 说明 | 举例 |
|------|------|------|
| 升 | 药性上升，助木气升达 | 桂枝、吴茱萸 |
| 降 | 药性下降，助金气收降 | 杏仁、半夏 |
| 浮 | 药性外散，助火气宣通 | 麻黄、薄荷 |
| 沉 | 药性内收，助水气封藏 | 熟地、五味子 |
| 中 | 药性归中，助中气旋转 | 炙甘草、干姜 |

### 治疗逻辑

```
症状 → 判断气机升降失常方向 → 确定中气状态 → 选药纠偏
```

1. **先看中气**：中气虚则必先补中，否则末端治疗无根
2. **再看升降**：升不及？降不及？升太过？降太过？
3. **选偏纠偏**：以药物的升降浮沉偏性，恢复圆运动

## 圆运动与辅行诀的区别

| 维度 | 辅行诀（omtp-core-fxj） | 圆运动（omtp-core-yyd） |
|------|------------------------|------------------------|
| 核心框架 | 五味体用化 | 升降浮沉圆运动 |
| 药物分类 | 按五味归五行（酸苦甘辛咸） | 按气机方向归类（升降浮沉中） |
| 组方逻辑 | 展开序列取药，体用化配伍 | 以偏纠偏，中气为先 |
| 剂量原则 | 君臣比例 3:1 | 只论比例，不给绝对剂量 |
| 适用场景 | 证型明确、单脏为主 | 多脏升降失调、中气枢纽失常 |

**两者不可混用**：同一次分析中只能使用一个理论框架。路由到 yyd 后，不再引用辅行诀体用化概念。

## 反模式

- 混用辅行诀体用化和圆运动升降——两套系统互斥
- 不先评估中气就直接治末端——违反"轴先轮后"原则
- 用苦寒直折代替降法——圆运动降法是顺其性降，非压制
- 忽略相火下行路径——相火不降是大量杂病的隐藏根因
