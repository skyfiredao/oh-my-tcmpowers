# 辅行诀五行五味五脏核心

本技能是辅行诀 skill 体系的**核心理论**。包含五行基础、体用展开序列、五行互含、体用化结构等所有分支规则共用的基础概念。不作为入口直接使用，由入口 skill 前置加载。

## 五行基础

### 五行序列与方向

木、火、土、金、水。此顺序为**顺时针**方向，反之为**逆时针**。

### 相生（母子关系，顺时针）

木 → 火 → 土 → 金 → 水 → 木

木生火，火生土，土生金，金生水，水生木。

《难经》云："母能令子虚，子能令母实。""虚则补其母，实则泻其子。"

### 相克

木 → 土 → 水 → 火 → 金 → 木

木克土，土克水，水克火，火克金，金克木。

### 相乘（克制太过）

沿相克方向，克的力量过强或被克者太弱。相克的异常亢进。

### 相侮（反克）

与相克方向相反，被克者反过来欺侮克者。

### 顺逆与补泻

顺时针为阳进，阳进为补，其数七，火数也。补药方用**七味药**。

逆时针为阴退，阴退为泻，其数六，水数也。泻药方用**六味药**。

化独立于补泻之外。

### 体用展开序列

五行按顺时针排列，每行内部用（补）在前、体（泻）在后，循环往复，共10个位置：

```
位置1:  木用（辛）  补
位置2:  木体（酸）  泻
位置3:  火用（咸）  补
位置4:  火体（苦）  泻
位置5:  土用（甘）  补
位置6:  土体（辛）  泻
位置7:  金用（酸）  补
位置8:  金体（咸）  泻
位置9:  水用（苦）  补
位置10: 水体（甘）  泻
→ 回到位置1
```

每个位置代表一个**味**（不是一味具体的药），从该味的5味药中选取一味入方。选哪一味由行中之行（目标脏腑）决定。

### 药物的双层味

每味药有两层味属性：

- **第一层味**：药物本身的味（五行互含表中的列）。例如代赭石味苦，苦属水。
- **第二层味**：由行中之行推导。行中之行对应的脏腑有自己的体味和用味，该味即为药物的第二层味。例如代赭石行中之行=木，木的体味=酸，故代赭石的第二层味含酸。

药物匹配展开序列上的位置时，**第一层味或第二层味均可用于匹配该位置的味**。第一层味优先匹配；当第一层味无法匹配目标位置时，以第二层味匹配。

### 展开序列取药规则

七和六既是药味数量，也是在展开序列上的取药范围。泻方和补方都是**体用交替**逐位取药，区别在于起始点的阴阳属性和方向：

- **泻方**（六味药）：从目标脏腑的**体位**起，**逆时针**数六个位置，逐位取药（体用体用体用）
- **补方**（七味药）：从目标脏腑的**用位**起，**顺时针**数七个位置，逐位取药（用体用体用体用）

泻方和补方的取药范围内体位和用位交替出现，因此方剂中**补泻药物搭配**，不是全补或全泻。

### 五行互含的作用机理

药物的两层五行坐标决定其作用路径和味属性：

- **第一层**（味属哪行）是药物的天然属性，决定它进入哪个脏腑。
- **第二层**（行中之行）决定它进入该脏腑后实际作用于哪个脏腑。行中之行同时赋予药物**第二层味**——该行的体味或用味。例如代赭石味苦属水（第一层），行中之行=木（第二层），木的体味=酸，故代赭石兼含酸味属性。

入口脏腑与目标脏腑之间的关系，就是五行生克关系本身。每味之"主"药（桂、旋覆、人参、五味子、地黄）味行与含行一致，进入本脏且作用于本脏，第一层味与第二层味相同。

五行生克是关系，不是力——没有方向阻力，没有衰减，没有传导效率差异。木生火就是生，水克火就是克，不存在"顺畅"或"困难"的区别。药力的强弱由药物之间的**配伍比例**调节，不由路径决定。

因此，组方选药的逻辑是：选味定入口，选行中之行定目标，以双层味匹配展开序列位置，调配伍比例定强度。五行关系决定"发生什么"，配伍比例决定"发生多少"。

## 君臣佐使

《经》云："主于补、泻者为君；数量同于君而非主，故为臣；从于佐监者，为佐、使。"

- **君**：主导方剂补泻方向的药
- **臣**：用量与君相同，但不主导方向
- **佐、使**：辅助和监制

君药决定方剂是补方（七味药）还是泻方（六味药）。

## 体用化三层结构

每一行内部分三层：

- **体**（阴）— 本体，物质基础，形质结构
- **用**（阳）— 功用，作用力，功能活动
- **化**（阴阳交互）— 阳作用于阴，化生出的状态

### 关键术语

- **虚**：衰弱、空，不足
- **补**：填充、裨益、补救
- **实**：充足、填满
- **泻**：消除、散去
- **苦**：怕、患、难以忍受、受累、为难（不指药味，指患者难受的主观状态）
- **化**：消解、消融、转变

### 辨治对应

| 层面 | 病理 | 治法 | 取味 |
|------|------|------|------|
| 用   | 虚（功能失调） | 补其不足 | 用之味 |
| 体   | 实（结构异常） | 泻其变异 | 体之味 |
| 化   | 苦（状态不佳） | 化其疲态 | 化之味 |

### 体用化动态循环

体、用、化三者之间存在闭合的相互制约与生化关系，其味所属五行遵循生克规律：

- **体克用**：结构限制功能。体之味所属五行克用之味所属五行。
- **用克化**：功能制约状态。用之味所属五行克化之味所属五行。
- **化生体**：状态影响结构。化之味所属五行生体之味所属五行。

以肝（木）为例：体味酸（金），用味辛（木），化味甘（土）。金克木（体克用），木克土（用克化），土生金（化生体）。长期坐姿不正（化/状态不佳）影响骨骼肌肉（体/结构），结构异常限制正常活动（用/功能），功能不畅又导致状态进一步恶化，循环往复。

### 化味的合成

化味有两种来源：

1. **直接化味**：方中存在化味药物，直接提供化味效果（效果=100%）
2. **合成化味**：方中同时存在某脏腑的体味药和用味药时，体+用合成出该脏腑的化味

合成化味对照：

| 脏腑 | 体味 | 用味 | 合成→化味 |
|------|------|------|----------|
| 肝 | 酸 | 辛 | 酸+辛→甘 |
| 心 | 苦 | 咸 | 苦+咸→酸 |
| 脾 | 辛 | 甘 | 辛+甘→苦 |
| 肺 | 咸 | 酸 | 咸+酸→辛 |
| 肾 | 甘 | 苦 | 甘+苦→咸 |

合成规则：

- 合成化味效果暂定为直接化味的50%以下
- 合成化味为一级合成，可与其他一级味按比例参与作用
- 二级合成（合成化味再与其他味合成）效果指数级递减，实际分析中可忽略

### 经文原文

**肝**：肝德在散。以辛补之，以酸泻之。肝苦急，急食甘以缓之，适其性而衰也。

**心（心包）**：心德在耎。以咸补之，苦泻之。心苦缓，急食酸以收之。

**脾**：脾德在缓。以甘补之，辛泻之。脾苦湿，急食苦以燥之。

**肺**：肺德在收。以酸补之，咸泻之。肺苦气上逆，急食辛以散之，开腠理以通气也。

**肾**：肾德在坚。以苦补之，甘泻之。肾苦燥，急食咸以润之，至津液生也。

## 体用化的五行推导公式

对于任一行 X：

1. **用味** = X 本行之味（顺其德性，补其功能）
2. **体味** = 克 X 之行的味（克我者主我之体，泻其形质）
3. **化味** = X 所克之行的味（我克者之味，化其不适）

### 推导过程

**木**：味辛皆属木，木的用之味为辛。金克木，味酸皆属金，故木的体之味为酸。木克土，味甘皆属土，故木的化之味为甘。

**火**：味咸皆属火，火的用之味为咸。水克火，味苦皆属水，故火的体之味为苦。火克金，味酸皆属金，故火的化之味为酸。

**土**：味甘皆属土，土的用之味为甘。木克土，味辛皆属木，故土的体之味为辛。土克水，味苦皆属水，故土的化之味为苦。

**金**：味酸皆属金，金的用之味为酸。火克金，味咸皆属火，故金的体之味为咸。金克木，味辛皆属木，故金的化之味为辛。

**水**：味苦皆属水，水的用之味为苦。土克水，味甘皆属土，故水的体之味为甘。水克火，味咸皆属火，故水的化之味为咸。

## 共药观察（待定）

## 通用推理步骤

本框架提供对应关系和结构，具体执行由上游 skill 驱动。以下为通用推理步骤参考：

1. **定脏**：确定涉及哪个脏腑（→ 确定所属五行）
2. **辨层**：判断是虚（功能）、实（结构）、还是苦（状态）
3. **取味**：根据体用化总表，确定对应治法和味
4. **验证**：用五行推导公式反向验证取味是否正确
5. **关联**：检查是否涉及相生、相克、相乘、相侮等跨行关系
