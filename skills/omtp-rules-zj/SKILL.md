## PART ONE：取穴规则（12 法）

### 1) 五俞穴取穴法

核心原则：井、荥、俞、经、合各有主病倾向，按病位与病性选取。

经典依据：

- 《灵枢·顺气一日分为四时》：“病在脏者取之井，病变于色者取之荥，病时间时甚者取之俞，病变于音者取之经，病在胃及以饮食不节得病者取之合。”
- 《难经·六十八难》：“井主心下满，荥主身热，俞主体重节痛，经主喘咳寒热，合主逆气而泄。”

| 五俞分类 | 主病要点（归纳） |
|---|---|
| 井 | 脏病、心下满、急证初起 |
| 荥 | 身热、热证、色变相关 |
| 俞 | 时作时甚、体重节痛 |
| 经 | 喘咳、寒热往来、音变相关 |
| 合 | 腑病、逆气、泄泻、饮食内伤 |

### 2) 子母补泻法

核心原则：虚则补其母，实则泻其子（《难经·六十九难》）。

先决规则（五俞五行配属）：

- 阴经：井木、荥火、俞土、经金、合水
- 阳经：井金、荥水、俞木、经火、合土

两种操作法：

1. 本经子母补泻：在“本经”五俞中取母穴或子穴。
2. 他经子母补泻：在“母经”或“子经”上取相应五行要穴。

示例：

- 肺经虚证（肺属金）：补母（土生金）。
  - 本经法：取肺经土穴（俞，太渊）。
  - 他经法：取脾经要穴以培土生金。
- 肝经实证（肝属木）：泻子（木生火）。
  - 本经法：取肝经火穴（荥，行间）。
  - 他经法：取心经/小肠经相关火性要穴以泄其子。

### 3) 表里经配穴法

核心原则：阴阳表里两经同取，以通调脏腑相关病机。

六组表里关系：

- 肺—大肠
- 脾—胃
- 心—小肠
- 肾—膀胱
- 心包—三焦
- 肝—胆

应用要点：病机同时涉及某脏及其相表里之腑，或经脉互见证候时优先使用。

### 4) 原络配穴法

又称主客配穴法。

核心原则：病经取原（主），其表里经取络（客）。经典依据见《灵枢·经脉》。

| 主经 | 主（原） | 客经 | 客（络） |
|---|---|---|---|
| 肺 | 太渊 | 大肠 | 偏历 |
| 大肠 | 合谷 | 肺 | 列缺 |
| 胃 | 冲阳 | 脾 | 公孙 |
| 脾 | 太白 | 胃 | 丰隆 |
| 心 | 神门 | 小肠 | 支正 |
| 小肠 | 腕骨 | 心 | 通里 |
| 膀胱 | 京骨 | 肾 | 大钟 |
| 肾 | 太溪 | 膀胱 | 飞扬 |
| 心包 | 大陵 | 三焦 | 外关 |
| 三焦 | 阳池 | 心包 | 内关 |
| 胆 | 丘墟 | 肝 | 蠡沟 |
| 肝 | 太冲 | 胆 | 光明 |

### 5) 俞募配穴法

核心原则：阴病行阳，阳病行阴；背俞与募常配合使用以增强调脏腑之效。

经典依据：《难经·六十七难》：“阴病行阳，阳病行阴。”

| 脏腑 | 背俞穴 | 募穴 |
|------|--------|------|
| 肺 | 肺俞 | 中府 |
| 心 | 心俞 | 巨阙 |
| 肝 | 肝俞 | 期门 |
| 脾 | 脾俞 | 章门 |
| 肾 | 肾俞 | 京门 |
| 心包 | 厥阴俞 | 膻中 |
| 胆 | 胆俞 | 日月 |
| 胃 | 胃俞 | 中脘 |
| 大肠 | 大肠俞 | 天枢 |
| 小肠 | 小肠俞 | 关元 |
| 膀胱 | 膀胱俞 | 中极 |
| 三焦 | 三焦俞 | 石门 |

### 6) 郄穴取穴法

核心原则：郄为气血深聚之处，多用于急性病与痛证。

关键法则：阴经郄主血证，阳经郄主急痛。

16 郄（十二经 + 奇经四郄）：

- 手太阴肺：孔最
- 手阳明大肠：温溜
- 足阳明胃：梁丘
- 足太阴脾：地机
- 手少阴心：阴郄
- 手太阳小肠：养老
- 足太阳膀胱：金门
- 足少阴肾：水泉
- 手厥阴心包：郄门
- 手少阳三焦：会宗
- 足少阳胆：外丘
- 足厥阴肝：中都
- 阴跷脉：交信
- 阳跷脉：跗阳
- 阴维脉：筑宾
- 阳维脉：阳交

经典依据：《针灸甲乙经》。

### 7) 八会穴取穴法

核心原则：脏、腑、气、血、筋、脉、骨、髓各有会穴，按组织/物质病变归类选穴。

经典依据：《难经·四十五难》。

- 脏会=章门
- 腑会=中脘
- 气会=膻中
- 血会=膈俞
- 筋会=阳陵泉
- 脉会=太渊
- 骨会=大杼
- 髓会=悬钟

应用要点：如血证多从血会（膈俞）入手，筋病多从筋会（阳陵泉）入手。

### 8) 下合穴取穴法

核心原则：合治内腑（《灵枢·邪气脏腑病形》）。

六腑下合：

- 胃→足三里
- 大肠→上巨虚
- 小肠→下巨虚
- 膀胱→委中
- 三焦→委阳
- 胆→阳陵泉

### 9) 八总穴/四总穴

四总穴歌：

肚腹三里留，腰背委中求，头项寻列缺，面口合谷收。

现代常用扩展为八总穴：

- 心胸取内关
- 小腹三阴谋（多指三阴交）
- 酸痛阿是穴
- 急救刺水沟

应用要点：按病位快速建立“区域指令穴”初始方案，再结合辨证加减。

### 10) 缪刺

核心原则：左病取右，右病取左；多取对侧络脉相关穴。

经典依据：《素问·缪刺论》。

鉴别：

- 缪刺：对侧络脉取穴
- 巨刺：对侧同名经（患经）取穴

应用：单侧病变、络脉病证。

### 11) 会穴取穴法

核心原则：多经会之穴可统调多经，适用于病机牵涉经脉较广者。

常用示例：

- 三阴交（足三阴会）
- 大椎（六阳经会）
- 百会（多经会）

### 12) 选穴原则

临床总则（三层并用）：

1. 近部取穴：局部就近取穴。
2. 远部取穴：循经远取（经脉所过，主治所及）。
3. 辨证取穴：按寒热虚实、脏腑经络病机定主次。
