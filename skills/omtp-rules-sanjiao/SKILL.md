# 温病三焦辨证规则

本技能包含三焦辨证的分层诊断规则。三焦辨证源自吴鞠通《温病条辨》，是温病的横向分区框架，以上焦-中焦-下焦为空间定位，与卫气营血的纵向深度分期互为补充。本 skill 为温病提供横向空间分区诊断，结论写入共享分析契约，作为独立诊断维度供下游参考。

本 skill 仅提供"病在哪一区"的诊断分区结论，不涉及任何选方逻辑。温病体系的用药结构尚未加入本系统。

## 前置条件

本 skill 无前置 skill 依赖。分区判断基于症状表现、脏腑定位和舌脉特征，不依赖五行互含体系。

## 三焦分区总则

吴鞠通《温病条辨》核心纲领：

> "上焦如雾，中焦如沤，下焦如渎。"
>
> 治上焦如羽，非轻不举；治中焦如衡，非平不安；治下焦如权，非重不沉。

三焦由上至下，反映温邪在人体的空间分布：

| 三焦 | 脏腑 | 病机特点 | 治则特点 |
|------|------|---------|---------|
| 上焦 | 肺、心（心包） | 温邪犯肺，或逆传心包 | 轻清宣透（"非轻不举"） |
| 中焦 | 脾、胃 | 热结阳明，或湿困太阴 | 平调寒热（"非平不安"） |
| 下焦 | 肝、肾 | 真阴耗损，虚风内动 | 填补真阴（"非重不沉"） |

## 各三焦诊断要点

### 上焦证

**主症**：发热恶寒，咳嗽，头痛，口渴，或高热神昏谵语（逆传心包）

**鉴别要点**：
- 病位在肺与心（心包），以呼吸系统和神志症状为主
- 轻证（肺卫）：发热咳嗽为主，神志清
- 重证（逆传心包）：高热伴神昏谵语，病情急剧

### 中焦证

**主症**：壮热面赤，腹满便秘（阳明热），或身热不扬，脘痞呕恶（太阴湿）

**鉴别要点**：
- 病位在脾胃，需分辨阳明热证与太阴湿证
- 阳明热：壮热面赤，腹满便秘（热象突出）
- 太阴湿：身热不扬，脘痞呕恶（湿象突出）

### 下焦证

**主症**：低热不退，手足心热，或手足蠕动，神倦耳聋，舌绛少苔

**鉴别要点**：
- 病位在肝肾，多见于温病后期
- 以真阴耗损为核心特征：低热不退、手足心热
- 虚风内动：手足蠕动，为阴虚至极之象

## 三焦传变规则

### 顺传

上焦 → 中焦 → 下焦（温邪由上而下，逐渐深入）

### 逆传心包

上焦肺卫之邪不经中焦直入心包（仍在上焦范围内，但病情急剧加重）

## 三焦与卫气营血的对应关系

两套辨证体系从不同维度描述温病进程，可交叉参照：

| 三焦 | 常见卫气营血分期 | 说明 |
|------|----------------|------|
| 上焦 | 卫分、气分（早期）、营分（逆传心包） | 上焦涵盖浅层（表）到深层（营）的跨度 |
| 中焦 | 气分 | 中焦证基本等同气分证的脾胃部分 |
| 下焦 | 营分、血分 | 下焦多见于温病后期阴液耗损 |

## 使用流程

1. 从 `shared-zhengzhuang-analyze.md` 读取症状分析结果（步骤 1-5 已完成）
2. 判断是否为温病（热性外感），若否则不使用本 skill
3. 按主症、脏腑定位判断当前所在三焦（上/中/下）
4. 可与卫气营血分期交叉参照，增强定位准确性
5. 将分区结论写入共享分析契约，作为独立诊断维度

## 反模式

- 将内伤方（五脏补泻）归入三焦辨证体系 -- 三焦辨证专论温病外感，内伤用脏腑辨证
- 跳过分区直接下结论 -- 必须先明确"病在哪一区"
- 将三焦辨证与卫气营血辨证简单等同 -- 两者维度不同（空间 vs 深度），交叉参照但不互相替代
- 用分区结论直接推导方剂 -- 温病用药结构尚未加入本系统，分区结论仅作为诊断维度
